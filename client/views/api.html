API

The Kramster REST API is open and entirely free! Make awesome apps using the Kramster question data. An idea: Multiplayer quiz app.

PS! All underscores in URL are replaced with spaces by the server before querying the database.

API Verb Description URL Return type (as JSON)
GET Return all documents http://kramster.it/api/documents [Object]
GET Return all documents of given school http://kramster.it/api/documents/:school [Object]
GET Return all documents of given school and course http://kramster.it/api/documents/:school/:course [Object]
GET Return a given number of random questions from given course of given school http://kramster.it/api/documents/:school/:course/random/:number [Object]
GET Return single document by school, course and document name http://kramster.it/api/documents/:school/:course/:document [Object]
GET Return array of all distinct schools http://kramster.it/api/list/schools [String]
GET Return array of all distinct courses at given school http://kramster.it/api/list/:school [String]
GET Return array of all distinct documents at given school and course http://kramster.it/api/list/:school/:course [String]

Document JSON schema


                {
                    _id: ObjectId,
                    school: String,
                    course: String,
                    name: String,
                    mode: String,
                    questions: [{
                        question: String,
                        options: [String],
                        answers: [Number]
                    }]
                }
