Contribute

Currently the API does not have POST methods available. This is coming later. Therefore, if you want to contribute to the database, you can do this by sending your question documents (exams) per email, and I will add it for you. What a great way to help your fellow students, don't you think?

If you're a hero, you can send me a .txt on the following format:

The AnswerIndex is the correct answer's index in the following list of options.

School
Course
Name
Question1 §§ AnswerIndex §§ Option1 §§ Option1 §§ ... §§ OptionN
Question2 §§ AnswerIndex §§ Option1 §§ Option1 §§ ... §§ OptionN
...
QuestionN §§ AnswerIndex §§ Option1 §§ Option1 §§ ... §§ OptionN

Example:

University of Foo, San Bar
FOO1337 Foobarian Algebra
Exam Spring 2014
What is foo? §§ 0 §§ Bar §§ Bob §§ Alice §§ Dijkstra
What is bar? §§ 1 §§ Pub §§ Foo §§ Chocolate
